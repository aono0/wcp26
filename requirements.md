# WCP26 要件定義書（最終版）

**バージョン**: 2.0  
**最終更新**: 2026-06-04  
**ステータス**: 実装完了（App Store審査待ち）

---

## 1. プロジェクト概要

### 1.1 目的
FIFAワールドカップ2026を楽しむサッカーファン向けに、試合情報・チームデータ・動画をワンストップで提供するiOSアプリ。

### 1.2 対象
- プラットフォーム: iOS 16以上
- 主要ターゲット: 日本語ユーザー
- リリース目標: 2026年6月11日（WC2026開幕日）

---

## 2. 実装済み機能一覧

### 2.1 タブ構成（5タブ）

| タブ | 画面名 | 主な機能 |
|------|--------|---------|
| 1 | ホーム | 直近試合・マイチーム・スタッツウィジェット |
| 2 | 動画 | YouTube Shorts フィード（TikTok風縦スクロール） |
| 3 | マイチーム（中央・強調） | チームフォロー・試合通知管理 |
| 4 | 試合・順位 | 試合日程・順位表・スタッツランキング |
| 5 | 出場国 | 48か国一覧・国詳細・選手情報 |

### 2.2 ホーム画面
- 直近試合カード（上位5件）
- マイチームセクション（お気に入りチームの次試合・フォロー追加ボタン）
- スタッツウィジェット（得点/アシストランキングTOP3、大会開幕後に表示）

### 2.3 動画タブ
- YouTube Shorts（60秒以下・埋め込み可能）の縦型フィード
- 垂直スクロールで次の動画に切り替え（TikTok風）
- 横型動画はサムネイル＋「YouTubeで見る」ボタン表示
- 3時間ごとに自動更新（2026年5月以降の動画のみ）

### 2.4 マイチームタブ
**「マイチーム」サブタブ**
- フォロー中チームの次試合・直近結果・グループ順位表
- 「次の試合」右端の「全日程を見る」で国詳細へ遷移

**「試合通知」サブタブ**
- フォロー中チームの試合を自動収録（登録不要）
- ベルボタンで個別試合の通知追加も可能
- 試合カード右上のベルアイコンでON/OFF

### 2.5 試合・順位タブ
**試合ビュー**
- グループステージ: 日付チップで絞り込み（MD別ではなく日付別）
- 決勝トーナメント: R32/R16/QF/SF/3位/決勝 切り替え
- 未確定カードは「グループA 1位 vs グループB 2位」形式で表示

**順位表ビュー**
- グループA〜L 全12グループ表示
- P/W/D/L/GF/GA/GD/Pts の完全テーブル
- 上位2チーム（決勝T進出）をグリーンで強調

**スタッツビュー**
- 得点ランキング / アシストランキングの切り替え
- 大会開幕後にリアルタイム更新

### 2.6 出場国タブ
- 48か国グループ別一覧（GROUP A〜L）
- 国詳細: 試合日程・結果・選手一覧（GK/DF/MF/FW別）
- 選手タップで選手詳細（背番号・クラブ・得点・アシスト）

### 2.7 認証・アカウント
- **現状（ローカル開発）**: モックログイン（dev-login）
- **本番**: Sign in with Apple（Apple Developer承認後に実装予定）
- アカウント登録フォームなし。Apple IDの匿名SubIDで識別
- アカウント削除機能（App Store必須要件）実装済み

### 2.8 Push通知
- 毎朝9:00 JST にフォロー中チームの翌日試合を検索して通知送信
- 試合単位でも個別に通知登録可能（ベルボタン）
- 通知ON/OFF設定あり

## 3. データ

### 3.1 初期データ
| データ種別 | 件数 | 更新方法 |
|-----------|------|---------|
| 出場国 | 48か国（グループA〜L） | 固定 |
| グループステージ試合 | 72試合 | football-data.org で自動更新 |
| 決勝トーナメント試合 | 31試合（プレースホルダー込み） | 対戦確定後に手動更新 |
| 選手データ | 1,248名（48か国×26名） | 固定（スタッツは自動更新） |
| YouTubeチャンネル | 20チャンネル以上 | Supabaseで管理 |

### 3.2 リアルタイム更新データ
| データ | 更新タイミング | ソース |
|--------|-------------|--------|
| 試合スコア・ステータス | 試合近辺5分ごと | football-data.org |
| グループ順位表 | 試合結果と連動 | DBから自動計算 |
| 得点/アシストランキング | 試合結果と連動 | football-data.org |
| YouTube動画 | 3時間ごと | YouTube Data API v3 |

## 4. 非機能要件

| 項目 | 仕様 |
|------|------|
| レスポンス速度 | APIレスポンス 1秒以内（gzip圧縮・DBインデックス最適化済み） |
| キャッシュ | TanStack Queryで国一覧30分・試合2分・動画15分 |
| セキュリティ | JWT認証・レートリミット・管理者エンドポイント保護 |
| データ保護 | pushToken・認証IDはAPIレスポンスに含めない |
